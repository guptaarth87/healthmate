API Documentation

Authentication

POST /auth/signup: Create a new user.

Request Body:

      
        {
          "name": "string",
          "age": "number",
          "gender": "string",
          "pincode": "string",
          "aadhaarNumber": "string",
          "phoneNumber": "string",
          "password": "string",
          "email": "string"
        }

Sample Request:

      
        POST /auth/signup
        Content-Type: application/json

        {
          "name": "John Doe",
          "age": 30,
          "gender": "Male",
          "pincode": "12345",
          "aadhaarNumber": "123456789012",
          "phoneNumber": "9876543210",
          "password": "password123",
          "email": "john@example.com"
        }

POST /auth/login: Authenticate user.

Request Body:

      
        {
          "aadhaarNumber": "string",
          "password": "string"
        }

Sample Request:

      
        POST /auth/login
        Content-Type: application/json

        {
          "aadhaarNumber": "123456789012",
          "password": "password123"
        }

Data Management

POST /data/adddata: Add new data.

Request Body:

      
        {
          "diseaseName": "string",
          "medicineName": ["string"],
          "doctorName": "string",
          "city": "string",
          "nextAppointment": "date",
          "refillDate": "date",
          "aadhaarNumber": "string"
        }

Sample Request:

      
        POST /data/adddata
        Content-Type: application/json

        {
          "diseaseName": "Fever",
          "medicineName": ["Paracetamol", "Ibuprofen"],
          "doctorName": "Dr. Smith",
          "city": "New York",
          "nextAppointment": "2024-03-15",
          "refillDate": "2024-03-10",
          "aadhaarNumber": "123456789012"
        }

DELETE /data/deletedata/:id: Delete data by ID.

GET /data/getalldata: Get all data records.

GET /data/getdatabyaadhaar/:aadhaarNumber: Get data by Aadhaar number.
